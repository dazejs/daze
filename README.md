## 介绍

`Daze.js` 是一款高性能、高扩展性、功能强大的 `Node.js` 服务端 Web 开发框架。

## 特性

- 基于 TypeScript 开发
- 面向切面编程（AOP）
- 提供 ORM、验证器、资源层、服务层、中间层等抽象层能力
- 基于 IOC 容器设计模式，模块高度解耦，支持依赖注入
- 高性能路由设计，性能优异
- 可通过服务提供者模式进行扩展，具有非常强大的扩展能力

## 快速开始

##### 安装工具

```bash
$ npm install -g @dazejs/cli
```

##### 创建工程

```bash
$ daze create example
$ cd example
$ npm start
```

访问 `http://localhost:8080` 进行预览
