## Introduction

`Daze.js` is an efficient, highly extensible and powerful `Node.js` server-side Web development daze.


## Installation

```bash
$ npm install -g @dazejs/cli
```


## Getting Started

```bash
$ daze create example
$ cd example
$ npm start
```

visit `http://localhost:8080` to preview
